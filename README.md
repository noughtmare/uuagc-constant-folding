# uuagc-constant-folding
Example of a constant folding transformation implemented with uuagc.

## How to run
```
$ git clone --recurse-submodules https://github.com/noughtmare/uuagc-constant-folding.git
$ cd uuagc-constant-folding
$ cabal run
```
